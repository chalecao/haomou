安装黑苹果mac10.9.2教程

发表于
Dec 31 2014

安装黑苹果

本篇讲的是安装黑苹果MAC 10.9.2系统，不过安装最新的Yosemite 10.10 系统应该也没问题，按照教程的步骤操作就可以了。我这里使用的是clover引导的，比变色龙好用，如果你的硬盘支持AHCI模式，建议使用clover引导。
先下载安装工具资源吧，本人使用acer 4738g的笔记本，安装所用的资源已经共享到百度网盘: [mac 10.9.2 资源包] (http://pan.baidu.com/s/1bn71bx9 )


文件列表：

  1. Mac 10.9.2 系统dmg
  2. config.plist 用于配置引导的
  3. transMac10.4注册版，用于读写上面的dmg 系统镜像的
  4. Kext Utility,用于安装驱动的
  5. VooDooPS2controller, ps2键盘和触摸板驱动，不过这个我没用到，装了系统之后已经驱动了
  6. VooDooHDA, 声卡驱动，这个不完美，装了之后有时有声音，有时没有
  7. BCM5722D, 无线网卡和只能电池驱动，完美
  8. ATI 6370m驱动，这个里面有好几个，我用的是3000的那个驱动，这个是我自己找的，之前别人用的是其他两个驱动，这个可以自己试一下。
    需要设备：8G以上u盘一个
    转载请注明出处：http://www.haomou.net/2014/12/31/2014_mac_install/

系统分区

一定要先做好系统分区，这个超级至关重要，为后边省下很多事情。推荐工具 diskgenius，这个软件可以自己下载。
这个是应该先做好的。但是放到后面再说，便于理解。

制作引导安装盘

现在打开Transmac，setup.exe安装，全部默认安装即可，安装完成以后用文件夹内提供的注册码（key.txt）注册即可，安装完成会自动在桌面生成transmac图标，然后以管理员身份运行Transmac，打开软件按照以下截图操作。

然后选择OS X系统镜像（mac 10.9.2.dmg），点击打开，系统会提示你需要格式化U盘，点击OK即可，然后就是等待进度条走完，那么你的OS X引导安装U盘就算制作完成了。

完成之后，系统会多出来两个盘，一个是usbboot引导盘，一个是mac系统安装盘。这个是写dmg镜像时候自动格式化的u盘。

安装mac

重启系统，注意需要将硬盘设置成AHCI模式才可以，否则mac系统认不出硬盘。然后从u盘进入系统，开机时按住f2设置boot顺序或者按住f10，然后选u盘也可以。

格盘分区

u盘引导后，会有两个选项，选第一个进入winPE，然后选择分区工具，将硬盘分出一个区来安装mac 系统，这个不要分区不要放到硬盘起始位置，因为hd(0,0)待会后面我们会做引导。在硬盘前面可以留1g的空间做引导。如下图所示，如果你是安装双系统，建议在硬盘末尾分区安装mac，这样偏于硬盘寻道。

分区方法：使用diskgenius，winpe中自带的有，先压缩现有硬盘分区，然后新建分区，注意新建之后不要格式化。然后给新建的分区分配一个驱动卷号，比如h，我们待会把mac安装到h盘中。

安装mac

重启，选择第二个clover引导，然后如下图类似：

选择第一个 Boot Mac OS X Install from Install OS X ,，如果前面一切顺利，你应该进入了以下界面，请选择磁盘工具，然后继续。
进入后不要着急安装，在实用工具一栏点选磁盘工具，先在左边磁盘列表里面选择需要安装Mac OS X的分区，选中之后一定确认大小是否和起初在Windows下分出来的磁盘一致，然后点击“抹掉”，格式选择：Mac OS 扩展(日志式)，名称随便填写，然后点击抹掉，然后系统会弹出确认窗口，点击抹掉以后，在窗口左上角点击红色圆形按钮（不是屏幕左上角）退出。
[在这里务必解释下，由于OS X与Win对于磁盘容量进位制不同，所以你可能会发现在磁盘工具格式化时候的磁盘比Win下你所格式化的容量偏大，这是正常的请勿怀疑]

选择安装OS X，相信如果你前面没错的话，应该可以进入下一步安装界面了。

若出现图片情况继续即可
等待吧，快装完了，最后一秒等的时间比较久，可能需要几分钟的时候，请耐心

设置引导

重启系统，如果拔出u盘，你就无法启动了，因为安装系统之后，更改了分区表，原来的活动分区变成了非活动分区，但是还是可以通过u盘引导进去的。
插上u盘，同样进入到winPE，然后建议将第一步在硬盘最开始，hd(0,0)也就是第一个主分区的第一个盘设置成引导盘，这个引导盘1G空间就够了。其实做成写dmg之后自动生成的usbboot那个盘就可以了，当然这里我们就不能再使用transmac了，因为我们要把引导做在硬盘上不是移动设备。这个可以通过diskgenius中的clone 分区的功能，注意要先将usbboot盘和本地硬盘的需要做引导的第一个盘设置成非活动盘才可以用。

将usbboot盘克隆到hd（0，0）那个盘就可以了。然后将克隆后的hd(0,0)设置成活动盘即可正常引导。
注意：这里面为什么非要用hd（0，0）呢，因为我们用的那个dmg是别人大师制作好的，大师在引导中clover的引导写死了是hd(0,0)，更改不了，所以必须用第一个主分区的第一块盘做引导盘。不信你可以试试哈。

安装驱动

恭喜你，你已经离成功不远了，不出意外，可以正常引导并进入到mac系统了。此时你可以升级系统，进入app store 可以免费升级系统到MAC 10.9.5，经测试没有什么问题。

然后就是安装驱动了，打开Kext Utility，然后稍等片刻，等软件初始化后，你可以将驱动文件，也就是后缀名为kext的文件拖动到软件上即可，然后再等等 出现complete！ 就是安装成功了。
1、 dsdt.zip

这个dsdt只修正了睡眠问题，就是删除了EHC1和EHC2相关代码其它什么也没改（唤醒需要按电源键）
dsdt放进\Extra\目录，没有请自己创建

2、 VoodooHDA-v2.8.4.zip
声卡，直接安装就可以了，遇到破音时，到其MIDI设置中，把编码改为16位就好，mic如没有请到偏好设置中最下面的Voodoohda设置，右边切换到 录音设置页面，调一下上面两个滑块HDMI声音无解

3、 ATI6370M.kext.zip
显卡是改了Ati5000Controller的Eulemur参数，状态栏透明，用aty_init驱动有水波纹，但没有那种一圈一圈的东西，用dsdt和变色龙驱动有，但睡眠唤醒后花屏，所以我选择了aty_init来驱动

4、 BCM5722D.kext.zip
网卡和显示电量驱动

5、 VoodooPS2Controller.kext.zip
触控板和键盘驱动

谢谢！
